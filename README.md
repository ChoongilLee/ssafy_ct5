## CT5_mis.py : 제가 에상하는 예시답안 code 실수

(6,1)
2 3 2 ->  2-(l6-2l+l3-1l)<0 -> 2 (원래는 0)
5 1 10 ->  10-(l6-5l+l1-1l)=7 -> 7
6 1 9 ->  9-(l6-6l+l1-1l)=9 -> 9
7 3 10 ->  10-(l7-6l+l3-1l) -> 9
